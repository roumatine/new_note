## 部分常用 `git` 指令
`git init` 初始化本地仓库
`git clone + <link>`
`git remote -v` 查看git分支和仓库别名
`git add .`
`git add + file.name` 把工作区中的代码提交到暂存区中。
`git status` 查看缓存区状态
`git rm --cached + file.name` 撤回缓存区文件
`git commit -m + "note" `  提交到本地库和备注，如果想指定文件，就在指令最后加文件名
`git push origin master` origin是仓库的别名，master 代表主分支，我们默认是在主分支里提交
